HTML es un lenguaje de estructura

Las siglas de HTML significan: HYPER TEXT MARKUP LANGUAGE Lenguaje muy sencillo, en su origen, que permitía consultar documentación científica por medio de enlaces o hyper textos

  • Hipertexto: Enlaces
  • Marcado: Etiquetas

Un documento de HTML se conforma de 3 etiquetas principales:

  1. La etiqueta de documento: DOCTYPE html
  2. La etiqueta de head
  3. La etiqueta de body
  4. página
